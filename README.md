# Final Project

This is the Final Project for the IBM AI Engineering course - an emotion detection application using Watson NLP.

## Features

- Course management with lessons
- Exam system with multiple-choice questions
- User authentication and authorization
- Admin interface for managing courses, questions, and submissions
- Exam grading with detailed result feedback

## Models

- **Course** - Course information
- **Instructor** - Course instructors
- **Learner** - Platform learners
- **Enrollment** - Links learners to courses
- **Lesson** - Course lessons
- **Question** - Exam questions linked to courses
- **Choice** - Multiple-choice options for questions
- **Submission** - Tracks learner exam submissions

## Tech Stack

- Python Django
- Bootstrap 5
- SQLite database
